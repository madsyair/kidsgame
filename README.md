# 🎮 Kids Game — Belajar Sambil Bermain

Koleksi game edukasi matematika gratis untuk anak TK-SD. Tanpa iklan, bisa offline.

🔗 **Main di sini:** [madsyair.github.io/kidsgame](https://madsyair.github.io/kidsgame/)

---

## 🎲 Daftar Game

### 🧮 MathCeria — Petualangan Hitung ✨ Baru

Game belajar matematika lengkap: **Tambah, Kurang, Kali, Bagi**. Dari konsep dasar sampai soal cerita sehari-hari.

🔗 **[Main MathCeria →](https://madsyair.github.io/kidsgame/mathceria/)**

**Fitur:**
- 🌐 Bilingual (Indonesia/English)
- 4 kategori × 5 level = **20 level**
- 🖼️ Visual emoji untuk membantu anak menghitung
- 📖 Soal cerita sehari-hari di level lanjut
- ⏱️ Timer adaptif (60s → 30s per level)
- 🔥 Combo bonus, 🔄 retry, 🏆 scoreboard
- 🔒 Unlock system: 10× sempurna → level baru
- 💾 Auto-save progress

**Struktur Level:**

| Kategori | Level 1-2 | Level 3 | Level 4 | Level 5 |
|----------|-----------|---------|---------|---------|
| ➕ Tambah | Visual detail (TK) | Visual simple (SD 1) | Angka saja (SD 2) | Soal cerita (SD 2-3) |
| ➖ Kurang | Visual detail (TK) | Visual simple (SD 1) | Angka saja (SD 2) | Soal cerita (SD 2-3) |
| ✖️ Kali | Kelompok visual (SD 1-2) | Kelompok simple (SD 2-3) | Angka saja (SD 3) | Soal cerita (SD 3-4) |
| ➗ Bagi | Bagi visual (SD 1-2) | Bagi simple (SD 2-3) | Angka saja (SD 3) | Soal cerita (SD 3-4) |

### 🦊✕ Kuis Perkalian — Kids Multiplication Quiz

Game interaktif belajar perkalian dengan wave system dan combo streak.

🔗 **[Main Kuis Perkalian →](https://madsyair.github.io/kidsgame/multiplication/)**

**Fitur:**
- 🌐 Bilingual (Indonesia/English)
- 🎮 5 level bertahap sesuai kurikulum SD
- 🌊 Wave system — soal bertambah setiap wave
- ⏱️ Skor berbasis waktu + combo streak
- 🏆 Papan skor & riwayat permainan
- 💀 Game over jika 3× gagal berturut atau AFK 5 menit

**Level:**

| Level | Nama | Rentang |
|-------|------|---------|
| 🌱 LV1 | Perkalian Mudah | ×1, ×2, ×5, ×10 |
| 🔷 LV2 | Tabel Kecil | 1-5 × 1-5 |
| 🔥 LV3 | Tabel Lengkap | 1-10 × 1-10 |
| 💎 LV4 | Puluhan | 10-90 × 2-9 |
| 👑 LV5 | Campuran | Semua acak |

## 🛠️ Teknologi

Semua game menggunakan stack yang sama:

- **Single HTML file** per game — tanpa build tools
- **Vanilla JavaScript** — tanpa framework/library
- **Web Audio API** — efek suara tanpa file eksternal
- **localStorage** — simpan progress
- **PWA-ready** — bisa di-install di HP
- **Responsive** — HP, tablet, desktop
