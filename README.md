# 🦊✕ Kuis Perkalian — Kids Multiplication Quiz

Game interaktif belajar perkalian untuk anak SD. Bilingual (Indonesia/English).

🔗 **Main di sini:** [https://madsyair.github.io/kidsgame/multiplication/](https://madsyair.github.io/kidsgame/multiplication/)

## ✨ Fitur

- 🎮 5 level bertahap sesuai kurikulum SD
- 🌊 Wave system — soal bertambah setiap wave
- ⏱️ Skor berbasis waktu + combo streak
- 🏆 Papan skor & riwayat permainan
- 💀 Game over jika 3× gagal berturut-turut atau AFK 5 menit
- 📶 Offline — bisa dimainkan tanpa internet
- 📲 PWA — bisa di-install di HP
- 🌐 Bilingual: Bahasa Indonesia & English
- 🔊 Efek suara & animasi

## 📚 Urutan Level

| Level | Nama | Rentang | Pedagogis |
|-------|------|---------|-----------|
| LV1 🌱 | Perkalian Mudah | ×1, ×2, ×5, ×10 | Perkalian paling intuitif |
| LV2 🔷 | Tabel Kecil | 1-5 × 1-5 | Hafalan tabel dasar |
| LV3 🔥 | Tabel Lengkap | 1-10 × 1-10 | Termasuk 7×8, 6×9 dll |
| LV4 💎 | Puluhan | 10-90 × 2-9 | Nilai tempat |
| LV5 👑 | Campuran | Semua acak | Fleksibilitas |

## 🛠️ Teknologi

- Single HTML file (self-contained)
- Vanilla JS — tanpa framework/library
- Web Audio API untuk efek suara
- LocalStorage untuk menyimpan progress
- PWA + Service Worker untuk offline
